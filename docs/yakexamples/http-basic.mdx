# 发送一个 HTTP 请求

## 使用 http 基础库

```go
rsp, err = http.Get("http://baidu.com")
die(err)

# 获取 StatusCode
printf("StatusCode: %v\n", rsp.StatusCode)

# 获取一个 Header
printf("Content-Length: %v\n", rsp.Header["Content-Length"])

# 生成完整数据包
rspRaw, err = http.dump(rsp)
die(err)

println(string(rspRaw))

```

执行上述代码运行结果为

